Prepare the Anchors

  1. Select 4 units to be anchors
  2. Mount them high on walls or tripods, preferably above head height
  3. Ensure they are all on the same plane
  4. Arrange them approximately in a rectangular shape
  5. Power them using USB mains power supplies or USB battery packs

Prepare the Tags

  1. Select 1 unit to be a tag
  2. Power the tag by either (a) connecting a USB battery pack or (b) opening the enclosure and inserting a rechargeable battery

Prepare the Tablet

  1. Download the .apk Android application file (Decawave DRTLS Manager) from the Decawave website to your tablet
  2. Follow the instructions on the tablet to install the application

Create a Network

  1. Open the Decawave DRTLS Manager application
  2. Tap ‘Start Device Discovery’
  3. Tap one of the discovered devices and an option will appear to name a network – this device will be added to that network
  4. Repeat for the remaining devices to add them to this network
  5. All units will initially appear as tags in ‘passive’ UWB mode

Configure the Units as Anchors or Tags

Tap the network and then tap each device in turn to configure it:
  1. First, identify the unit by unchecking the LED checkbox and tap accept button in upper right corner to save. The LEDs on that device will turn off.
  2. To configure the 4 anchors:
    1. Set the ‘Node Type’ to ‘Anchor’
    2. Set ‘UWB’ to ‘Active’
    3. For any one of the anchors, check the ‘Initiator’ box (there must always be one initiator in the network)
    4. Tap accept button in upper right corner to save
  3. To configure the tag:
    1. Set the ‘Node Type’ to ‘Tag’
    2. Set ‘UWB’ to ‘Active’
    3. Set the location rate to a fast rate be selecting: ‘100 ms/10 Hz’
    4. Select ‘responsive mode’, ‘stationary detection’ and ‘location engine’
    5. Tap accept button in upper right corner to save

Auto-Position the Anchors (for up to 4 anchors)

 Note: Using the Auto-Positioning function to automatically retrieve anchor locations is a demonstration-only feature. For a commercial installation this feature should not be used. Accurately measured anchor positions should be entered manually (see below).
  1. From the ‘Network Details’ screen, tap the “Auto-Position” button in the pull-down menu on the upper right
  2. A list of anchors (within Bluetooth range) appears
  3. Re-order the anchors in the list to match their locations in the room:
    1. Order the anchors to be anti-clockwise in the room
    2. The 1st anchor in the list is the (0,0) co-ordinate
    3. The 2nd anchor is to the right of the 1st (on the x-axis)
    4. The 3rd anchor should be positioned ‘above’ the 2nd, as it is viewed on the grid
    5. The 4th anchor should be positioned ‘above’ the 1st anchor
  4. Tap “Measure” to start the auto-positioning
  5. Preview locations before saving by tapping ‘PREVIEW’
  6. Enter heights of the anchors by tapping ‘Z-AXIS’
  7. Save the anchors setup by tapping ‘SAVE’

Manually Position the Anchors

 The co-ordinates of the anchors can be measured and entered manually in the settings screen of each anchor.
  1. From the ‘Network Details’ screen, edit each anchor by tapping
  2. Enter the x,y,z co-ordinate in the ‘position’ section
  3. Tap accept button in the upper right corner to save
